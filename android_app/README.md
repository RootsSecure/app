# NRI Plot Sentinel Android App

Kotlin Android app scaffolded with MVVM, Retrofit, Gson, ViewModel, and LiveData.

## Update Before Running

Open [build.gradle.kts](D:/RootsSecure_app/android_app/app/build.gradle.kts) and replace:

- `BASE_URL`
- `API_HOST`
- `CERT_PIN`

with your real backend values.

## Project Structure

```text
android_app/
  app/
    src/main/java/com/rootssecure/nriplotsentinel/
      api/
      repository/
      ui/
      viewmodel/
```

## Launch

Open `android_app` in Android Studio and run the `app` configuration.
